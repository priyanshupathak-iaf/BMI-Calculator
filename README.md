# BMI Calculator 🧮

A simple, responsive, and user-friendly **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**. This project helps users quickly calculate their BMI based on height and weight and understand their health category.

---

## 🚀 Features

- 📏 Input height and weight
- ⚡ Instant BMI calculation
- 📊 Displays BMI result with category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- 🎨 Clean and responsive user interface
- 🌙 Smooth user experience with basic validations

---

## 🛠️ Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling and layout (responsive design)  
- **JavaScript** – Logic and calculations  

---

## 📖 How It Works

1. User enters height (in cm or meters) and weight (in kg)
2. Clicks the **Calculate** button
3. JavaScript calculates BMI using the formula:

   BMI = weight (kg) / (height (m)²)

4. The result is displayed along with the health category

---

## 🧠 BMI Categories

| BMI Range       | Category       |
|----------------|--------------|
| Less than 18.5 | Underweight  |
| 18.5 – 24.9    | Normal       |
| 25 – 29.9      | Overweight   |
| 30 and above   | Obese        |
